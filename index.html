Get started in 2 minutes

1

Clone the repo

git clone https://github.com/victordelrosal/agentic-archetypes.git

2

Pick your archetype

Choose the one that matches your business type: freelancer, content creator, remote worker, e-commerce, or local service.

3

Fill in config.yaml

Your name, services, rates, tools, preferences. 60 seconds. This personalizes everything.

4

Open Claude Code

cd freelancer && claude

The system detects it's your first run and walks you through onboarding. You're operational.
